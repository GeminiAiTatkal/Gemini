How It Works: Quick Setup Guide

Watch our detailed video tutorial for a step-by-step installation and usage guide.

1

Download Extension

Get the **Gemini** extension and the Activator for your browser (Chrome/Edge).

2

Run Activator

Double-click the activator file. This helps Chrome/Edge trust the extension. (Restart browser if needed).

3

Install Extension

Drag and drop the downloaded `.crx` file onto your Chrome/Edge extensions page (chrome://extensions).

4

Start Booking!

Open IRCTC, fill your details in the extension, and enjoy super-fast bookings!
